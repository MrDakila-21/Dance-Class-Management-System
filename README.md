# Dance-Class-Management-System
A full-stack R Shiny web application for dance studio operations, featuring class scheduling, customer bookings, QR-based attendance tracking, and real-time analytics. Built with R, SQLite, and JavaScript, the system includes automated class archiving, slot management, and role-based authentication.

Key Features:
- Class scheduling with instructor assignments
- Customer booking system with member discounts
- QR code attendance verification
- Revenue tracking and performance analytics
- Automated data archiving and reporting
